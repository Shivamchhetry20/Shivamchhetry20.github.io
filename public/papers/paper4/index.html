üöÄ Building a CNN Model in PyTorch

üìå Introduction

In this blog, we will implement a Convolutional Neural Network (CNN) using PyTorch to classify images. The architecture consists of:

  • Two convolutional layers with ReLU activation
  • Max-pooling layers for feature extraction
  • A fully connected layer for classification

üèóÔ∏è Model Architecture

Below is a summary of the CNN model:

Layer Type Output Shape
Input 1x28x28 28x28x1
Conv2D 3x3, 8 filters 28x28x8
ReLU Activation 28x28x8
MaxPool 2x2 14x14x8
Conv2D 3x3, 16 filters 14x14x16
ReLU Activation 14x14x16
MaxPool 2x2 7x7x16
Flatten - 1x784
Fully Connected Linear num_classes

üõ† Implementing CNN in PyTorch

import torch

# üèóÔ∏è Defining a CNN Model in PyTorch
class PyTorchModel(torch.nn.Module):
    def __init__(self, num_classes):
        super().__init__()
        
        self.num_classes = num_classes  # üéØ Define number of output classes

        # üß© Feature Extractor: Convolutional Layers + Activation + Pooling
        self.features = torch.nn.Sequential(
            torch.nn.Conv2d(1, 8, kernel_size=3, stride=1, padding=1),  # üèóÔ∏è Conv Layer 1
            torch.nn.ReLU(),  # ‚ö° Activation Function
            
            torch.nn.MaxPool2d(kernel_size=2, stride=2),  # üîΩ Downsampling
            
            torch.nn.Conv2d(8, 16, kernel_size=3, stride=1, padding=1),  # üèóÔ∏è Conv Layer 2
            torch.nn.ReLU(),  # ‚ö° Activation
            
            torch.nn.MaxPool2d(kernel_size=2, stride=2)  # üîΩ Downsampling
        )

        # üèÅ Fully Connected Output Layer
        self.output_layer = torch.nn.Linear(7*7*16, num_classes)  # üìä Final classification layer

    def forward(self, x):
        x = self.features(x)  # üîÑ Forward pass through CNN
        x = torch.flatten(x, start_dim=1)  # üìù Flatten tensor for FC layer
        x = self.output_layer(x)  # üéØ Compute logits (raw scores)
        return x
